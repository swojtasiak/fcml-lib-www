Download

You can download the project using the direct links below or choose the preferred distribution archive from the list of the files accessible through the SourceForge site. If you would like to build the library on your own, choose the distribution archive. If you use Windows and you are not interested in building the binaries yourself, choose "Windows pre-built binaries" archive. It contains the libraries built for Visual Studio as well as for MinGW. The binary files are compiled for both x86 and x86_64 architectures.
